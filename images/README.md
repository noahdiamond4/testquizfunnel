# Report page images

Drop the five generated images into this folder with these exact
filenames (JPG format). The page hides any section's image gracefully
if its file is missing, so partial uploads are safe.

| Filename             | Used in                              | Aspect |
|----------------------|--------------------------------------|--------|
| `hero.jpg`           | Report hero background               | 4:5    |
| `limescale.jpg`      | "What We Found" findings card        | 3:2    |
| `absorption.jpg`     | Science section (skin/lungs/hair)    | 1:1    |
| `product-black.jpg`  | Offer card (black-fixture visitors)  | 4:5    |
| `product-chrome.jpg` | Offer card (chrome-fixture visitors) | 4:5    |
| `filter-compare.jpg` | Proof card (new vs. used cartridge)  | 3:2    |

Easiest upload: open this folder on github.com → "Add file" →
"Upload files" → drag all six in → commit.

Keep each file under ~250 KB (the hero) / ~200 KB (the rest) for
fast mobile loads. tinypng.com works well for compression.
